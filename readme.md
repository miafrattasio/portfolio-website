# Mia Frattasio | Professional Portfolio & Resume Site

This repository hosts the static website portfolio for myself, a Computer Science student focusing on Cybersecurity, with experience in application security, web development, and financial technology.

The site is designed to provide a comprehensive view of my education, work history, projects, and technical skills.

## Key Features

* **Responsive Design:** Fully optimized for desktop, tablet, and mobile viewing (implemented via CSS media queries).
* **Themed Layout:** Utilizes a clean, professional color theme for a consistent visual identity.
* **Clear Navigation:** Separate pages for different content types: Home/About, Projects, Experience, and Other Info.

## Project Structure

The portfolio is built using standard HTML and CSS.

| File/Folder | Description |
| :--- | :--- |
| `index.html` | **Home Page:** Contains the main profile, contact links, About Me, Education, and Core Skills sections. |
| `projects.html` | **Projects Page:** Detailed descriptions of key technical projects (e.g., WPIBuys, Juice Shop Penetration Testing). |
| `experience.html` | **Experience Page:** Detailed listing of all work and internship roles. |
| `other.html` | **Other Info Page:** Lists Honors, Awards, and Activities/Leadership roles. |
| `styles.css` | **Styling:** Contains all custom CSS, including global variables and responsive layout rules. |
| `profile.jpg` | Placeholder for the professional profile photo. |

## Technologies Used

* **HTML5:** Structure and content.
* **CSS3:** Styling and layout (`flexbox`, `grid`, and media queries).
* **Custom Variables:** Defined in `:root` for easy color management (`--primary-color`, `--text-color`, etc.).
* **Font:** Roboto (via Google Fonts or system fallbacks).
